前言

关于Hexo

Hexo是一种博客框架，类似于大名鼎鼎的wordpress。但是Hexo凭借其简单的操作方法以及和Github相配合的快速部署能力而受到许多博主的欢迎，目前市场上也已经有许多开发者提供的免费Hexo主题模板供大家选择，可以说Hexo是目前市面上兼顾了美观、操作、实用为一体的博客写作框架。



我曾经尝试过市面上主流的博客框架和平台：Wordpress，typecho，Hugo等等，但是都因为诸多原因类似于难上手、界面不够美观自己修改太过于麻烦等原因而停止博客的操作。但最后其实发现，博客的本质在于内容，形式和美观只是表层。最终自己选择了Hexo，也让自己沉下心，安安静静地写一些东西，给自己，也是给世界。

关于写作

我高中时候语文就一直不好，从小到大也是阅读多于写作，进入大学学了理科以后更加没有练习写作，加强输出的机会和习惯。但是进入社会，进入职场后发现，系统性的输出能力是非常重要的职场和生活能力。能否将自己的经验、感悟和积累整理成系统性知识并且加以输出，很多时候成为职场上评判能力的一个重要维度。试想，同样的项目结果,小A在项目结束后只是按部就班地完成了一份项目总结，做了一次公开复盘，但是小B在项目临近尾声时，和项目干系人进行了充分沟通，输出了一份完整、系统的项目总结和公开复盘，同时还从自己个人角度，输出一份个人总结报告和项目改进建议。这样的表现，一方面充分展现小B积极思考、对项目投入充分热情，对于小B的职业生涯发展也有非常大的助力。

以上例子只是想说明，优秀的、系统化的输出能力对于生活、职业甚至自己内心的宁静都有所助益，所以希望自己在新的一年，能够在写作和输出层面加强，提升自己的输出和表达能力。
